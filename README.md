# 📦 Warehouse Stock & Sales Analysis Dashboard

## 🚀 Overview
An interactive Power BI dashboard designed to provide real-time insights into warehouse operations, monitoring inventory levels across different floors and analyzing sales performance.

## 🛠️ Key Features
- **Stock Visualization:** Integrated a Synoptic Panel to visualize stock distribution on a warehouse floor plan.
- **KPI Monitoring:** Real-time tracking of Total Sales, Capacity, and Current Stock levels.
- **Data Modeling:** Cleaned and structured data from Excel to create a cohesive data model.
